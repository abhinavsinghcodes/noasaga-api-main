📌 Noasaga API

A free anime API to fetch anime episodes, seasons, and more.

📌 Base URL

https://noasaga-api-main.onrender.com

📜 Endpoints

🔹 Get All Anime Data

Endpoint: GET /anime/data

🔹 Get Anime by ID

Endpoint: GET /anime/:anime_id

Example: GET /anime/67898

🔹 Get Specific Season

Endpoint: GET /anime/:anime_id/:season_id

Example: GET /anime/67898/s1

🔹 Get Specific Episode

Endpoint: GET /anime/:anime_id/:season_id/:episode_number

Example: GET /anime/67898/s1/1

📌 How to Use

  • Use GET /anime/data to fetch all anime data.
  • Use GET /anime/:anime_id to fetch a specific anime.
  • Use GET /anime/:anime_id/:season_id to fetch a specific season.
  • Use GET /anime/:anime_id/:season_id/:episode_number to fetch a specific episode.
